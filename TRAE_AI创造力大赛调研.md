# TRAE AI 创造力大赛调研

调研日期：2026-06-22。赛事时间均按北京时间。

## 一句话结论

这是一个面向全民的 AI 创新实践赛：用 TRAE Work 或 TRAE IDE，把一个创意做成可体验的 AI 作品。报名、初赛、复赛都在 TRAE 官方中文社区发帖完成，决赛是线下路演。报名与初赛已经开始，报名截止为 2026-07-15 23:59。

## 赛制

- 阶段：报名、初赛、复赛、决赛。
- 参赛资格：面向所有人开放；未成年人需监护人同意；报名和初赛阶段仅限个人参赛。
- 工具要求：全程使用 TRAE Work 或 TRAE IDE，国际版 / 中国版均可；需保留能证明创作过程的材料。
- 赛道：4 个通用赛道四选一，另有 1 个附加赛题可叠加。
- 通用赛道：生活娱乐、学习工作、社会服务、硬件交互。
- 附加赛题：社会公益，可在通用赛道基础上叠加，决赛获奖时奖励可兼得。
- 初赛晋级：最多 350 个复赛名额，专业评审通道 300 个，抖音人气通道 50 个。
- 复赛晋级：最多 20 个决赛名额。
- 决赛：线下路演，评选决赛奖项。

## 要做什么

报名阶段要提交创意提案帖：

- 标题：报名赛道 + 创意名称。
- 标签：生活娱乐 / 学习工作 / 社会服务 / 硬件交互四选一；如参加公益附加赛题，再加社会公益。
- 正文不少于 100 字，至少写清创意名称与介绍、目标用户与痛点、价值与意义。
- 必须附 TRAE Work 生成的创意产物 HTML 文件。

初赛阶段要提交可体验 Demo：

- 把报名创意做成可运行、可交互、能体验核心功能的 Demo。
- Demo 帖需包含 Demo 简介、创作思路、体验地址或 HTML 文件。
- 硬件交互赛道可以用公开视频替代在线体验。
- 需要展示 TRAE 实践过程：开发关键步骤截图不少于 3 张，关键任务 Session ID 不少于 3 个。
- 初赛作品有效前提是报名审核已经通过。

初赛评分维度：

- 创新性 30%：需求创新、解决思路、技术创新。
- 实用性 30%：场景洞察、解决效果、落地可行性。
- 完成度 20%：核心链路、用户体验、价值传递。
- 美观度 / 设计体验 20%：非硬件赛道看视觉交互体验；硬件赛道看硬件与软件系统协同能力。

## 怎么报名

1. 用同一个手机号注册 / 登录 TRAE 中国版和 TRAE 官方中文社区，并全程使用同一个社区账号。
2. 进入报名专区：https://forum.trae.cn/c/38-category/39-category/39
3. 按模板发一篇创意提案帖，上传 TRAE Work 生成的创意产物 HTML 文件。
4. 等官方审核。官方每个工作日审核报名帖，一般提交后次日审核周期内可看到结果，周末和节假日顺延。
5. 审核通过后，登录官网领奖入口确认领取：https://www.trae.cn/ai-creativity/result
6. 报名通过后即可进入初赛，把创意做成 Demo，并到初赛专区提交：https://forum.trae.cn/c/38-category/40-category/40

## 奖励概览

- 报名通过奖：不限名额，TRAE 中国版速通 Pro 月卡价值 99 元 + 决赛现场门票；需在官网确认领取。
- 初赛参与奖：2,000 名，大赛专属礼包价值 100 元。
- 复赛晋级奖：最多 350 名，TRAE 中国版速通 Pro+ 月卡、导师沟通机会、高曝光传播资源等。
- 复赛完赛奖：最多 350 名，定制周边礼包、火山引擎加速器 5 万元 AI 及云资源代金券等。
- 决赛现金奖池：113 万元；单个作品最高可得 35 万元现金。

## 关键注意事项

- 社区发帖不等于报名成功，必须等官方审核通过。
- 报名成功后还要到官网确认领取奖励，不会自动到账。
- 可以报名多个赛道 / 多个创意，但每个创意需要单独提交报名帖，报名通过奖励同一选手只发一次。
- 报名审核失败，不建议修改旧帖等复审；官方说明建议重新发一篇新创意帖。
- 报名创意和后续 Demo / 作品的核心方向必须一致，可以迭代细节但不要换题。
- 初赛专业评审晋级时，同账号多个作品只取最高分 1 个。
- 社区附件限制：支持文字、图片、GIF、链接、20M 内文件；大文件、视频、代码库先上传第三方平台，再贴公开链接。
