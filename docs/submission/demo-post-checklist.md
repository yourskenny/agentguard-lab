# AgentGuard Lab 初赛提交检查清单

- 报名主题与 Demo 保持一致：AgentGuard Lab，AI Agent 可靠性评测与修复工作台。
- 体验入口可以运行：`npm install`、`npm run dev`、浏览器打开 Vite 输出的本地地址；默认是 `http://127.0.0.1:5173/`，端口被占用时按终端提示为准。
- Demo 能展示完整闭环：Agent 配置编辑、测试集生成、Agent 输出编辑、评测运行、失败证据、失败归因、修复补丁、配置 diff、复测对比、报告预览。
- 帖子说明包含创作目标：把 Agent 从“能演示”推进到“可测试、可解释、可维护、可交付”。
- 帖子说明包含 TRAE 使用过程：主题规划、架构图、TDD、前端实现、浏览器验证。
- 准备至少 3 张过程截图：架构图、TDD 测试通过、工作台运行效果。
- 准备至少 1 段演示动线：选择课程问答 Agent，编辑系统提示词，生成测试集，修改一条 Agent 输出，运行评测，应用修复并复测，展示 diff，复制或下载报告。
- 验证最终门禁：`npm test` 和 `npm run build` 均通过。
- 确认公开仓库 README、架构图和运行方式可被评委直接查看。
